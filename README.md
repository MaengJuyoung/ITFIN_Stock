1. 사용 API
  ㄴ 공공데이터포털의 openAPI사용
  ㄴ 금융위원회_KRX상장종목정보 사용

2. 통신방법 자유
  ㄴ 서버 구축x, 스파이더젠만 사용

3. 버전
  ㄴ 4.1.3

4. 기능

  1) 종목조회
    ㄴ 조회조건: 종목코드, 종목명 2개만
    ㄴ 종목코드는 단축코드(likeSrtnCd)를 의미 / 종목명(likeItmsNm)은 포함

  2) 관심그륩
    ㄴ 그룹 추가, 삭제, 그룹명 변경 가능
    ㄴ 그룹 최소 1개 존재, (Default: 관심그룹001)

  3) 관심종목
    ㄴ 관심종목 추가, 삭제, 여러개 삭제, 전체 삭제

  4) 데이터
    ㄴ localStorage로 관리
